## Equipment Parser
1. Copy equipment files in xml folder
    ```
    cia_disguises.xml
    firearms_cia.xml
    firearms_pistols.xml
    firearms_pistols_cia.xml
    firearms_rifles.xml
    firearm_ammo.xml
    firearm_scopes.xml
    various.xml
    ```
2. Run `ts-node retrieveConfigData.ts`
3. New data will be written to `equipmentData.json`
